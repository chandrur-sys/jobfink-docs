Final Approval Process After Revision

Step 1: Open Course with Revision Status

  1. Navigate to the Course list.
  2. Select the course with status Revision.
  3. Click on the course to open it.

Step 2: Open Session Information

  1. Navigate to the Session Information tab.
  2. Verify that the default status selection is set to Approved such as.

Step 3: Approve the Course

  1. Click on the Approved button.
  2. Confirm the action if a confirmation popup is displayed.
  3. Once the Maßnahme is approved, the status is updated to Approved.
  4. The Approved button becomes disabled.

  5. The following fields are enabled:

    Maßnahme Certificate Number
    Start Date
    End Date
    Recertification Trigger

Step 5: View Available Locations

  1. Click on the Select Location dropdown.
  2. System displays locations based on the company mapped with the certification. k Run Mßnahme system generates a location-specific Maßnahme.

Expected Result:
A New Location specific Mßnahme is created based on the selected location.

Step 6: System Processing

  1. The system will validate the approval details.
  2. The approval will be saved successfully.

Step 7: Status Update

  1. Verify that the course status is updated to Approved.
  2. The approved status will be reflected in the course list and course details screen.

Expected Result

  • Default status is set to Approved.
  • Course is approved successfully.
  • Course status changes to Approved.
  • Approval details are saved in the system.
