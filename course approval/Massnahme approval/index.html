Maßnahme Approval Module

Overview

This page explains how to access the Maßnahmenzulassung screen in the OneHermes system after login.

The Maßnahme Approval module is used to Create A Global Maßnahme and Location speacific Maßnahme

Navigation Flow

After successful login, follow the steps below to open the Maßnahme Approval page:

Login → Certification → Maßnahme Approval → Global Maßnahme Screen

Step-by-Step Process

Step 1: Login to the System

  1. Open the OneHermes application.
  2. Enter valid username and password.
  3. Click Sign In.
  4. Dashboard page will be displayed.

Step 2: Click Certification Tab

  1. On the left sidebar, locate Certification.
  2. Click on the Certification menu.
  3. Sub-menu options will be displayed.

Available options: Carrier & Location Approval,Company, Maßnahmenzulassung

Step 3: Click Maßnahmenzulassung

  1. From the Certification sub-menu, click Maßnahmenzulassung.
  2. System will load the Maßnahmenzulassung screen.

Step 4: View Global Maßnahme List

After clicking Maßnahmenzulassung, the Global Maßnahme screen will be displayed, showing both Global Maßnahme and location-specificMaßnahme.

Search and Filter

Search

User can search Maßnahmes using the search box.

Filters

There are two filters available on the Summary screen.

1. Status Filter (Top Right Corner)

  1. Located at the top right corner of the screen.
  2. Used to filter Maßnahmes based on their status. ---->Example statuses:
    • Approved
    • Active
    • Awaiting Approval
    • Revision etc...

2. Maßnahme Type Filter (Maßnahme / location-specific Maßnahme)

  1. Available in the Maßnahme Creation filter section.
  2. Used to filter Maßnahmes based on Maßnahme type. ----> Users can filter Maßnahmes as:
    • Global Maßnahmes
    • Location-Specific Maßnahmes

This helps users to easily view and manage Maßnahmes based on location and category.

  • Expected Result

Maßnahme Approval page is opened successfully. Global Maßnahme list is displayed. User can view and manage Maßnahmes.
