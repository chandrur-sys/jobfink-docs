Location SpecificMaßnahme

Overview

After the Maßnahme is approved, the system allows users to select a target location.
The available locations are displayed based on the company linked to the selected certification.

Once a location is selected, a location-specific Maßnahme is automatically created for the Maßnahme.

Prerequisites

  • Maßnahme must be in Approved status.
  • Certification must be assigned to the Maßnahme.
  • Company should be mapped with locations.

Step-by-Step Process

Step 1: Verify Maßnahme Approval

  1. Login to the application.
  2. Navigate to Maßnahme Management.
  3. Open the required Maßnahme.
  4. Confirm that the status is Approved.

Expected Result:
Maßnahme status should be Approved.

Expected Result:
Only relevant company locations should be displayed.

Step 4: Select Location

  1. Choose the required location from the list.
  2. Save the selection.

Expected Result:
Selected location should be saved successfully.
