Measure Number Receipt and Status Update Process

Overview

After applying for a measure, the government authority responds with a Measure Number via email.

Once the response is received, the user must manually update the Maßnahme status and enter the received measure number in the system.

After saving, the system updates the Maßnahme status automatically.

Prerequisites

  • Maßnahme must be in Applied for Measure Number status.
  • Measure number response must be received via email.
  • User must have edit permission for Maßnahme details.

Step-by-Step Process

Step 1: Receive Measure Number Email

  1. Monitor the registered email inbox.
  2. Receive the response from the government authority.
  3. Note the received Measure Number.

Expected Result:
Measure Number is received through official communication.

Step 2: Open Maßnahme Details

  1. Login to the application.
  2. Navigate to Maßnahme Management.
  3. Open the relevant Maßnahme.

Expected Result:
Maßnahme details page is displayed.

Step 3: Update Maßnahme Status Manually

  1. Click on Edit or Update Status.
  2. Change the Maßnahme status to Measure Number Received.

Expected Result:
Status field becomes editable.

Step 4: Enter Measure Number

  1. Locate the Measure Number field.
  2. Enter the received Measure Number correctly.

Expected Result:
Entered value is accepted without validation errors.

Step 5: Save Changes

  1. Click the Save button.

Expected Result:
- Measure Number is stored successfully. - Maßnahme status is updated to Measure Number Received.

Validation Points

  • Measure Number format is validated.
  • Status cannot be updated without entering a valid Measure Number.
  • Changes are saved correctly.
  • Updated status is reflected in Maßnahme list view.

Exception Scenarios

  • If Measure Number is invalid, system displays error message.
  • If Save fails, status remains unchanged.
  • If user does not have permission, edit option is disabled.

Notes

  • Always verify the Measure Number before saving.
  • Keep email confirmation for audit reference.
  • Contact administrator for access-related issues.
